# 📊 图形化监控仪表板

## 新功能亮点

系统现已支持完整的图形化监控界面！

### ✨ 主要特性

#### 1. 实时监控仪表板
- 📈 **柱状图**：实时显示服务器 CPU、内存、磁盘使用率
- 📊 **统计卡片**：监控目标、告警、在线/离线状态一目了然
- 🔄 **自动刷新**：每30秒自动更新数据
- 🎨 **状态标识**：颜色编码快速识别正常/异常状态

#### 2. 详细监控页面
- 📉 **历史趋势图**：折线图展示资源使用历史趋势
- 🥧 **使用率饼图**：直观显示当前资源占用比例
- ⏱️ **时间范围选择**：支持1/6/12/24小时数据查看
- 📋 **监控记录表**：详细的历史监控数据列表

---

## 🚀 快速开始

### 访问仪表板
```
http://localhost:8080/dashboard
```

### 查看监控详情
1. 访问监控目标页面：http://localhost:8080/targets
2. 点击任意监控目标的"详情"按钮
3. 查看详细的图表和历史数据

## 🎨 图表类型

### 柱状图 (Bar Chart)
- **位置**：仪表板
- **用途**：对比多个指标
- **特点**：并排显示，易于比较

### 折线图 (Line Chart)
- **位置**：详情页面
- **用途**：显示历史趋势
- **特点**：平滑曲线，观察变化

### 饼图 (Doughnut Chart)
- **位置**：详情页面
- **用途**：显示占比关系
- **特点**：圆环样式，直观美观

### 日常监控
```
1. 打开仪表板

3. 浏览服务器图表
```

```
1. 选择24小时时间范围
2. 观察资源使用模式
3. 识别高峰和低谷时段
4. 制定优化方案
```

## 🔧 配置说明

### 监控间隔
在"系统配置"页面设置检查间隔（默认60秒）

### 告警阈值
在"系统配置"页面设置各资源的告警阈值：
- CPU 使用率阈值：80%
- 内存使用率阈值：80%
- 磁盘使用率阈值：80%

### 自动刷新
- 仪表板：30秒
- 详情页面：30秒
- 可手动点击刷新按钮

## 🆘 常见问题

**Q: 图表不显示？**
A: 检查是否有监控数据，等待60秒后刷新页面

**Q: 数据不更新？**
A: 检查监控目标是否启用，查看服务器日志

**Q: 趋势图显示不完整？**
A: 选择较短的时间范围，或等待采集更多数据

**Q: 如何导出数据？**
A: 当前版本暂不支持，可以查看数据库文件
