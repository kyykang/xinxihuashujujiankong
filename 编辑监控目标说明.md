# 编辑监控目标功能说明

## 功能概述

现在你可以编辑已创建的监控目标，修改配置参数而无需删除重建。

## 使用方法

### 1. 进入监控目标页面
访问：http://localhost:8080/targets

### 2. 编辑监控目标
在监控目标列表中，找到要编辑的目标，点击"编辑"按钮

### 3. 修改配置
在弹出的编辑窗口中，你可以修改：
- **名称**：监控目标的显示名称
- **类型**：监控类型（服务器、存储、应用、数据库、业务指标）
- **配置参数**：根据类型显示不同的配置项
- **状态**：启用或禁用该监控目标

### 4. 保存修改
点击"保存修改"按钮，系统会立即应用新配置

## 各类型可编辑的配置

### 服务器监控
- 服务器类型（本地/远程）
- 远程服务器配置：
  - 服务器地址
  - SSH端口
  - 用户名
  - 认证方式（密码/SSH密钥）
  - 密码或密钥文件路径
  - 磁盘路径
  - 进程名称
- 监控项（CPU、内存、磁盘）
- 启用/禁用状态

### 存储监控
- 存储路径
- 启用/禁用状态

### 应用监控
- URL地址
- 启用/禁用状态

### 数据库监控
- 主机地址
- 端口
- 用户名
- 密码（留空则不修改）
- 数据库名
- 启用/禁用状态

### 业务指标监控
- 数据库类型
- 主机地址
- 端口
- 用户名
- 密码（留空则不修改）
- 数据库名
- 查询SQL
- 告警阈值
- 启用/禁用状态

## 注意事项

1. **密码字段**：编辑时，密码字段显示为"留空则不修改"，如果不需要修改密码，可以留空
2. **立即生效**：修改保存后，下一次监控检查将使用新配置
3. **禁用监控**：如果暂时不需要某个监控，可以将状态改为"禁用"，而不必删除
4. **类型变更**：修改类型后，配置字段会自动更新为对应类型的字段

## 操作示例

### 示例1：修改远程服务器的IP地址
1. 点击服务器监控目标的"编辑"按钮
2. 修改"服务器地址"字段
3. 点击"保存修改"

### 示例2：临时禁用某个监控
1. 点击监控目标的"编辑"按钮
2. 将"状态"改为"禁用"
3. 点击"保存修改"
4. 该监控将停止检查，但配置保留

### 示例3：修改数据库密码
1. 点击数据库监控的"编辑"按钮
2. 在"密码"字段输入新密码
3. 点击"保存修改"

### 示例4：调整业务指标的SQL查询
1. 点击业务指标监控的"编辑"按钮
2. 修改"查询SQL"字段
3. 调整"告警阈值"
4. 点击"保存修改"

## 其他操作

### 查看监控数据
点击"查看"按钮可以查看该监控目标的最近监控数据

### 删除监控目标
点击"删除"按钮可以永久删除该监控目标及其历史数据

## 快捷键提示

- **ESC**：关闭编辑窗口
- **Enter**：在表单中按回车可快速提交（部分浏览器）

## 故障排查

### 编辑后监控不工作？
1. 检查配置参数是否正确
2. 查看告警记录是否有错误信息
3. 确认监控状态是"启用"
4. 检查服务器日志

### 无法保存修改？
1. 检查必填字段是否都已填写
2. 检查网络连接
3. 刷新页面重试
4. 查看浏览器控制台是否有错误

---

**提示**：修改配置后，建议查看告警记录，确认监控正常工作。
