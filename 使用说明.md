# 信息化运维监控系统使用说明

## 安装步骤

1. 安装Python依赖：
```bash
pip install -r requirements.txt
```

2. 启动系统：
```bash
python app.py
```

3. 访问系统：
打开浏览器访问 http://localhost:5000

## 功能说明

### 1. 监控仪表板
- 查看监控目标总数
- 查看告警统计
- 查看系统运行状态

### 2. 监控目标管理
支持添加以下类型的监控目标：

#### 服务器监控
- CPU使用率监控
- 内存使用率监控
- 磁盘使用率监控
- 网络IO监控
- 进程监控

#### 存储监控
- 存储容量使用率监控
- 支持自定义监控路径

#### 应用监控
- HTTP/HTTPS服务健康检查
- 响应时间监控
- 应用进程存活检测

#### 数据库监控
- MySQL连接状态监控
- 支持自定义数据库配置

#### 业务指标监控
- 支持MySQL和SQL Server
- 自定义SQL查询
- 设置告警阈值

### 3. 告警记录
- 查看所有告警历史
- 告警状态管理
- 告警信息详情

### 4. 系统配置
- 监控间隔设置
- 告警阈值配置
- 企业微信Webhook配置

## 企业微信告警配置

1. 在企业微信群聊中添加机器人
2. 获取Webhook地址
3. 在系统配置页面填入Webhook地址
4. 保存配置

告警消息将自动推送到企业微信群聊。

## 配置示例

### 添加服务器监控
1. 进入"监控目标"页面
2. 点击"添加监控目标"
3. 选择类型：服务器监控
4. 勾选需要监控的项目（CPU、内存、磁盘）
5. 保存

### 添加应用监控
1. 进入"监控目标"页面
2. 点击"添加监控目标"
3. 选择类型：应用监控
4. 输入URL地址（如：http://example.com）
5. 保存

### 添加数据库监控
1. 进入"监控目标"页面
2. 点击"添加监控目标"
3. 选择类型：数据库监控
4. 填写数据库连接信息
5. 保存

### 添加业务指标监控
1. 进入"监控目标"页面
2. 点击"添加监控目标"
3. 选择类型：业务指标监控
4. 选择数据库类型
5. 填写数据库连接信息
6. 输入查询SQL（如：SELECT COUNT(*) FROM orders WHERE status='pending'）
7. 设置告警阈值
8. 保存

## 注意事项

1. 确保系统有足够的权限访问被监控的资源
2. 数据库监控需要正确的连接信息和权限
3. 企业微信Webhook地址需要保密
4. 建议根据实际情况调整监控间隔和告警阈值
5. 定期检查告警记录，及时处理异常情况
